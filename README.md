# balance-teeter-totter

Demo: https://vue-balance-teeter-totter.herokuapp.com/

### Initialization
```bash
git git@github.com:AndreiSoroka/balance-teeter-totter.git;
cd balance-teeter-totter;
npm install;
```
### Build
```bash
// dev server
npm run serve;

// build
npm run build;

// buid for heroku
npm run heroku-postbuild;
npm start;
```


### Lints and fixes files
